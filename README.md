# RayTracer

A software ray-tracer following the [Ray Tracing in a weekend tutorial]{https://raytracing.github.io/books/RayTracingInOneWeekend.html#overview}

##  Tracers Available

* Software Ray Tracer (WIP)
* Vulkan Ray Tracer (TODO)

## Build and Run

    cmake -S . -B build
    cmake --build build
    ./build/src/hello
